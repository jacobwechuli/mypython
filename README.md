# MyPython - Data Science & AI Notebooks Repository

A collection of Jupyter notebooks demonstrating various Python techniques for data science, AI, and machine learning tasks.

## 📚 Repository Structure

This repository is organized into the following categories:

### 🤖 AI & LLM Applications
Notebooks focused on working with Large Language Models, prompt engineering, and AI-powered applications:

- **`001_prompt_evals.ipynb`** - Prompt evaluation techniques and best practices
- **`001_thinking_complete.ipynb`** - Advanced thinking patterns with LLMs
- **`005_hybrid.ipynb`** - Hybrid approaches combining multiple AI techniques
- **`006_web_search_complete.ipynb`** - Integrating web search with LLM applications
- **`RAG_Powered_Chatbot.ipynb`** - Building chatbots with Retrieval-Augmented Generation (RAG)
- **`PDF_Retrieval_and_Optimization_with_LlamaIndex.ipynb`** - PDF processing and retrieval using LlamaIndex

### 📊 Data Processing & Cleaning
Notebooks for data cleaning, preprocessing, and transformation:

- **`text_cleaning.ipynb`** - Text preprocessing and cleaning techniques
- **`wechuli_Loan_Data_Cleaning.ipynb`** - Real-world loan data cleaning example
- **`Wechuli_Working_with_JSON_Files.ipynb`** - JSON file handling and manipulation
- **`Chunking_Demo.ipynb`** - Text and data chunking strategies for processing large datasets

### 🖼️ Image Processing
Notebooks focused on image manipulation, processing, and analysis:

- **`Introduction_to_Python_Libraries_for_Image_Processing.ipynb`** - Overview of popular image processing libraries (OpenCV, Pillow, scikit-image, etc.)
- **`Image_Cleaning.ipynb`** - Image preprocessing and cleaning techniques

### 🔍 Embeddings & Vector Operations
Notebooks demonstrating embeddings and vector-based operations:

- **`Wechuli_Embeddings.ipynb`** - Working with embeddings, similarity search, and vector operations

## 🚀 Getting Started

### Prerequisites
- Python 3.7+
- Jupyter Notebook or JupyterLab
- Required libraries (see individual notebooks for dependencies)

### Installation

```bash
# Clone the repository
git clone https://github.com/jacobwechuli/mypython.git
cd mypython

# Install dependencies (recommended to use a virtual environment)
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install required packages
pip install jupyter notebook
```

### Running Notebooks

```bash
jupyter notebook
```

Then navigate to your desired notebook and run it.

## 📖 Notebook Descriptions

### AI & LLM Applications
- **Prompt Evals**: Learn how to evaluate and optimize prompts for better LLM performance
- **Thinking Complete**: Explore advanced reasoning patterns and chain-of-thought techniques
- **Hybrid Approaches**: Combine multiple AI techniques for complex problem-solving
- **Web Search Integration**: Enhance LLMs with real-time web search capabilities
- **RAG Chatbot**: Build intelligent chatbots using retrieval-augmented generation
- **PDF Retrieval with LlamaIndex**: Process and retrieve information from PDF documents efficiently

### Data Processing & Cleaning
- **Text Cleaning**: Remove noise, normalize text, handle special characters
- **Loan Data Cleaning**: Real-world example of preparing financial data for analysis
- **JSON Handling**: Read, parse, and manipulate JSON data structures
- **Chunking Demo**: Split large datasets into manageable chunks for processing

### Image Processing
- **Image Libraries Introduction**: Comprehensive guide to popular Python image processing libraries
- **Image Cleaning**: Preprocessing techniques including resizing, filtering, and normalization

### Embeddings
- **Embeddings**: Work with vector representations, similarity search, and semantic analysis

## 🛠️ Technologies & Libraries Used

- **LLMs & AI**: OpenAI API, LangChain, LlamaIndex
- **Data Processing**: Pandas, NumPy
- **Image Processing**: OpenCV, Pillow (PIL), scikit-image
- **Search & Retrieval**: Web search APIs, vector databases
- **JSON**: Python's built-in `json` library
